# n8n SDLC Framework

A portable SDLC framework for managing n8n workflows with dev/prod separation.

This README will be replaced with your project README when you run the import process.

## Quick Start

1. Install into your project: `./install.sh /path/to/your-project`
2. Open in Cursor or Claude Code
3. Say "get started" or run `/n8n-get-started`

The import process will auto-generate a project-specific README with your workflow inventory, architecture diagram, and development instructions.

For full framework documentation, see [SDLC Reference](n8n-sdlc/docs/SDLC-Reference.md).
